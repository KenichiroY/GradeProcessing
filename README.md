# 成績処理ソフト

小学校の先生向けのテスト成績管理ツールです。テストの点数を入力するだけで、教科ごとの集計やABC評価を自動で行います。

## 使い方

### 準備

1. `成績処理.xlsm` をExcelで開きます
2. 「マクロを有効にする」を選択してください
3. **1ファイル＝1クラス×1評価期間**（1学期、前期など）で運用します

### 初期設定（Settingシート）

最初に以下を登録しておきます。

| 列 | 内容 | 例 |
|----|------|----|
| A列 | 教科の頭文字（キー採番用） | J（国語）, S（算数） |
| B列 | 教科名 | 国語, 算数, 理科 |
| D列 | 評価の観点 | 知識・技能, 思考・判断・表現 |
| F列 | カテゴリ | テスト, 小テスト, 実技 |
| H-I列 | ABC評価の閾値 | A/B境界: 80, B/C境界: 50 |

### 児童名簿の登録（名簿シート）

- 11行目以降に児童のコード・姓・名を入力します
- F8セルに児童数が表示されます
- 「採番」ボタンでコードを自動付番できます
- 転出児童はF列（在籍終了日）に日付を入力します

### テストの登録

#### 1. 入力シートに情報を入力

| セル | 内容 |
|------|------|
| D4 | 教科 |
| F4 | カテゴリ |
| J4 | 実施日 |
| D6 | テスト名 |
| 8行目 | 評価の観点 |
| 12行目 | 配点 |
| 31行目～ | 児童ごとの得点 |

- 最大5つの観点（列）を同時に登録できます
- 得点欄に「-」を入力すると免除（評価対象外）になります

#### 2. 「登録」ボタンで確定

データシートにテストキー（J001, S002 など）が自動採番され、得点が転記されます。

### 得点の調整（オプション）

入力シートのチェックボックスで以下の調整が可能です。

- **クリッピング**: 得点の上限・下限を設定
- **得点変換**: 平方根変換または対数（底2）変換
- **範囲調整**: 調整後の得点範囲を指定

### 未入力データの管理（MENUシート）

1. 「未入力検索」ボタンで全テストの未入力一覧を表示
2. 一覧上で点数を直接入力
3. 「転記」ボタンでデータシートに一括反映

### 教科別の集計（Subjectシート）

1. 教科名を選択し、評価したい観点にチェックを入れます
2. 「追加/更新」ボタンでデータシートから該当テストを収集します

#### 重み正規化

配点が異なるテスト（50点満点と100点満点など）を同じ重要度で扱いたい場合、「重み正規化」ボタンを押します。基準配点（100点）に対する比率で重みが自動調整されます。

### ABC評価

1. Subjectシートで「評価」ボタンを押します
2. Settingシートに登録した閾値パターンが●マークで表示されます
3. 採用したいパターンの●をダブルクリックすると★に変わります
4. A計・B計・C計が自動集計されるので、適切な閾値を選びます
5. 「最終決定」をダブルクリックすると、Resultシートに転記され、スナップショットが保存されます

### データの修正

- **得点の修正**: データシートの得点セルをダブルクリック → 修正フォームが表示されます
- **テスト情報の修正**: データシートのヘッダー行（4～22行）をダブルクリック → 編集フォームが表示されます
- **テストの削除**: 編集フォームの「削除」ボタンから行えます

### 追試機能

テスト登録時、または登録後のテスト編集フォームから追試を設定できます。

1. 追試を設定すると、別ファイル（`成績処理_追試.xlsm`）に追試シートが作成されます
2. 追試中のテストはデータシート上でオレンジ色に着色されます
3. 追試シートで追試得点を入力し、算出方法を選択して最終得点を計算します
4. 「追試完了」で最終得点が本体ファイルに反映されます

算出方法は6種類から選べます：

| 方法 | 説明 |
|------|------|
| 合格点 | 本試で合格なら本試得点、追試で合格なら合格点、それ以外は最高点 |
| 最大値 | 全回の最高点 |
| 平均値 | 全回の平均 |
| 中央値 | 全回の中央値 |
| 内分点 | α×最高点＋(1-α)×本試得点 |
| 本試のみ | 追試結果を無視して本試の得点を使用 |

### テスト比較分析（IndividualAnalysisシート）

Analysisフォームから、同一教科のテストを2グループに分けて比較できます。箱ひげ図・散布図・T検定の結果が表示されます。

### 評価結果の確認（Resultシート）

全教科・全観点のABC評価結果がまとめて表示されます。列見出しはファイルを開いた時にSettingシートから自動生成されます。

### スナップショット

ABC評価を最終決定するたびに、そのときのSubjectシートの状態が `ファイル名_確定.xlsx` に自動保存されます。教科・観点ごとの評価履歴として活用できます。

## 得点の入力ルール

| 入力値 | 意味 |
|--------|------|
| 数値（0～配点） | 得点 |
| `-`（ハイフン） | 免除（評価対象外。達成率の計算から除外されます） |
| 空欄 | 未入力（MENUシートの未入力検索で検出されます） |

## ファイル構成

| ファイル | 説明 |
|----------|------|
| `成績処理.xlsm` | メインファイル（すべてのマクロを含む） |
| `成績処理_追試.xlsm` | 追試ファイル（追試設定時に自動生成） |
| `成績処理_確定.xlsx` | スナップショット（最終決定時に自動生成） |

## シート一覧

| シート名 | 用途 |
|----------|------|
| MENU | 未入力データの一覧と一括入力 |
| 名簿 | 児童名簿の管理 |
| テスト入力 | テスト得点の入力フォーム |
| データ | 全テストデータの蓄積（自動更新） |
| Subject | 教科別の集計とABC評価 |
| Result | 評価結果の一覧 |
| IndividualAnalysis | テスト比較分析 |
| Setting | 教科・観点・閾値の設定 |

## 注意事項

- シートやセルは保護されています。得点の修正はダブルクリックで表示されるフォームから行ってください
- データシートを直接編集しないでください
- 次の評価期間に移る際は、新しいファイルを作成してください
- 縄跳び等の上限がないデータは、最大値または目標値を配点として入力してください
