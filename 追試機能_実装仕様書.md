# 追試機能 実装仕様書

## 1. 概要

### 1.1 目的

既存の成績処理システム（`成績処理.xlsm`）に追試（再テスト）機能を追加する。
追試の進行管理を専用ファイルで行い、完了時に本体ファイルへ結果を反映する2ファイル構成とする。

### 1.2 全体構成

```
成績処理.xlsm           （本体ファイル：既存）
成績処理_追試.xlsm       （追試ファイル：新規作成）
```

- 本体ファイルから追試ファイルを自動生成する
- 追試ファイルはテストごとに1シート作成される
- 追試完了後、追試ファイルのボタン操作で本体ファイルに結果を反映する

### 1.3 基本フロー

```
[本体] テスト入力シートで「追試あり」にチェック
  │
  ├→ [本体] データシートに通常通り転記（得点セルは全員 "N" を設定）
  │
  ├→ [追試] 追試ファイルが存在しなければ新規作成
  │
  └→ [追試] 新しいシートを作成し、本試の得点と児童一覧を転記
         │
         ├→ [追試] 追試回を追加していく（追試1, 追試2, ...）
         │
         ├→ [追試] 全児童の最終得点が確定
         │
         └→ [追試] 「完了」ボタン押下
                │
                └→ [本体] データシートの "N" を最終得点で上書き
```

### 2.3 PostingModule.bas の変更

#### 2.3.1 Posting() への追試分岐追加

既存の `Posting()` プロシージャの末尾、データ転記処理（`Call TransferData`）の直後に追試処理を追加する。

```vb
' ========================================
' 追試処理（追試チェックボックスがONの場合）
' ========================================
If IsRetestEnabled() Then
    ' 検証：算出方法が選択されているか
    validationResult = ValidationModule.ValidateRetestSettings()
    If Not validationResult.IsValid Then
        Call ErrorHandlerModule.ShowValidationError(validationResult.ErrorMessage)
        GoTo CleanExit
    End If

    ' データシートの得点を全て "N" で上書き
    Call MarkAsRetestPending(numTest, lastRowData, lastColData)

    ' 追試ファイルへの転記
    Call RetestModule.CreateRetestSheet(numTest, lastRow, lastColData)
End If
```

#### 2.3.2 MarkAsRetestPending（PostingModule.bas に追加）

```vb
'===============================================================================
' データシートの得点セルを追試中マーカー "N" で上書き
' 説明：追試ありのテスト登録時、全児童の得点を "N" に置換する
'       TransferData で一旦生データを書き込んだ後に呼び出す
' 引数：
'   numTest - 今回登録したテスト数（列数）
'   lastRowData - データシートの児童データ最終行
'   lastColData - TransferData呼び出し前のデータシート最終列
'===============================================================================
Private Sub MarkAsRetestPending(ByVal numTest As Long, ByVal lastRowData As Long, _
                                 ByVal lastColData As Long)
    Dim i As Long, j As Long
    Dim targetCol As Long

    With Sh_data
        For i = 1 To numTest
            targetCol = lastColData + i
            For j = eRowData.rowChildStart To lastRowData
                ' 空欄と "-"（免除）はそのまま残す
                If Trim(.Cells(j, targetCol).Value & "") <> "" And _
                   .Cells(j, targetCol).Value <> "-" Then
                    .Cells(j, targetCol).Value = RETEST_MARKER
                End If
            Next j
        Next i
    End With
End Sub
```

### 2.4 ValidationModule.bas の追加

```vb
'===============================================================================
' 追試設定の検証
'===============================================================================
Public Function ValidateRetestSettings() As validationResult
    Dim result As validationResult
    Dim method As String
    Dim param As Variant

    result.IsValid = True

    ' 算出方法の確認
    method = Trim(sh_input.Range("算出方法セル").Value & "")
    If method = "" Then
        result.IsValid = False
        result.ErrorMessage = "追試の算出方法を選択してください。"
        ValidateRetestSettings = result
        Exit Function
    End If

    ' 有効な算出方法かチェック
    If method <> RT_METHOD_MAX And _
       method <> RT_METHOD_INTERPOLATION And _
       method <> RT_METHOD_CAPPED Then
        result.IsValid = False
        result.ErrorMessage = "算出方法「" & method & "」は無効です。" & vbCrLf & _
                              "「最大値」「内分点」「追試上限付き」から選択してください。"
        ValidateRetestSettings = result
        Exit Function
    End If

    ' パラメータが必要な方法のチェック
    param = sh_input.Range("パラメータセル").Value

    Select Case method
        Case RT_METHOD_INTERPOLATION
            If Trim(param & "") = "" Then
                result.IsValid = False
                result.ErrorMessage = "内分点方式ではパラメータ（α値：0～1）が必要です。"
                ValidateRetestSettings = result
                Exit Function
            End If
            If Not IsNumeric(param) Then
                result.IsValid = False
                result.ErrorMessage = "パラメータは数値で入力してください。"
                ValidateRetestSettings = result
                Exit Function
            End If
            If CDbl(param) < 0 Or CDbl(param) > 1 Then
                result.IsValid = False
                result.ErrorMessage = "内分点のα値は0～1の範囲で入力してください。" & vbCrLf & _
                                      "（1に近いほど最大値寄り、0に近いほど本試寄り）"
                ValidateRetestSettings = result
                Exit Function
            End If

        Case RT_METHOD_CAPPED
            If Trim(param & "") = "" Then
                result.IsValid = False
                result.ErrorMessage = "追試上限付き方式ではパラメータ（上限点）が必要です。"
                ValidateRetestSettings = result
                Exit Function
            End If
            If Not IsNumeric(param) Then
                result.IsValid = False
                result.ErrorMessage = "上限点は数値で入力してください。"
                ValidateRetestSettings = result
                Exit Function
            End If
            If CDbl(param) <= 0 Then
                result.IsValid = False
                result.ErrorMessage = "上限点は0より大きい値を入力してください。"
                ValidateRetestSettings = result
                Exit Function
            End If
    End Select

    ValidateRetestSettings = result
End Function
```

### 2.5 「N」マーカーへの既存処理の対応

以下の既存処理に「N」マーカーを考慮した変更を加える。

#### 2.5.1 統計値の数式（PostingModule.TransferData 内）

AVERAGE / MEDIAN / STDEV.P をAVERAGEIF系に変更する。
「N」と「-」を除外して計算するようにする。

```vb
' 変更前：
Sh_data.Cells(eRowData.rowAverage, targetCol).Formula = _
    "=AVERAGE(" & colLetter & eRowData.rowChildStart & ":" & colLetter & lastRowData & ")"

' 変更後：
Sh_data.Cells(eRowData.rowAverage, targetCol).Formula = _
    "=AVERAGEIFS(" & colLetter & eRowData.rowChildStart & ":" & colLetter & lastRowData & _
    "," & colLetter & eRowData.rowChildStart & ":" & colLetter & lastRowData & ",""<>N""" & _
    "," & colLetter & eRowData.rowChildStart & ":" & colLetter & lastRowData & ",""<>-"")"
```

> **注：** MEDIAN と STDEV.P には直接的なIF版がないため、配列数式を使うか、追試完了後の再計算で対応する。追試完了時（RetestModule.CompleteRetest 内）で「N」が全て上書きされた後に再計算が走るので、追試中のテストの統計値が一時的に不正確になることは許容する。

#### 2.5.2 SubjectModule.CollectSubjectData

「N」が含まれる列（追試未完了のテスト）をSubject収集対象から除外する。

```vb
' データシートから該当データを抽出するループ内に追加：
' 追試中チェック（"N" が1つでもあれば除外）
If HasRetestMarker(i) Then
    GoTo NextColumn    ' この列はスキップ
End If
```

```vb
'===============================================================================
' 指定列に追試中マーカー "N" があるか確認
' 引数：colIndex - データシートの列番号
' 戻り値：True = "N" が1つ以上ある（追試未完了）
'===============================================================================
Private Function HasRetestMarker(ByVal colIndex As Long) As Boolean
    Dim j As Long
    Dim childCount As Long

    HasRetestMarker = False
    childCount = sh_namelist.Range(RNG_NAMELIST_CHILDCOUNT).Value

    With Sh_data
        For j = eRowData.rowChildStart To eRowData.rowChildStart + childCount - 1
            If CStr(.Cells(j, colIndex).Value) = RETEST_MARKER Then
                HasRetestMarker = True
                Exit Function
            End If
        Next j
    End With
End Function
```

> **この関数は SubjectModule.bas に配置する。**

既存の空欄チェックロジックは変更不要（「N」は空欄ではないため、自動的にスキップされる）。

## 3. RetestModule.bas（新規モジュール — 本体ファイル内）

本体ファイル側に新規標準モジュールとして作成する。
追試ファイルの生成・シート作成・結果反映をこのモジュールで一元管理する。

### 3.1 追試ファイルのパス取得

```vb
'===============================================================================
' 追試ファイルのフルパスを取得
' 戻り値：追試ファイルのパス（例："C:\...\成績処理_追試.xlsm"）
'===============================================================================
Public Function GetRetestFilePath() As String
    Dim fso As Object
    Set fso = CreateObject("Scripting.FileSystemObject")

    Dim baseName As String
    Dim parentFolder As String

    baseName = fso.GetBaseName(ThisWorkbook.FullName)
    parentFolder = fso.GetParentFolderName(ThisWorkbook.FullName)

    GetRetestFilePath = parentFolder & "\" & baseName & RETEST_FILE_SUFFIX & RETEST_FILE_EXT

    Set fso = Nothing
End Function
```

### 3.2 追試ファイルの生成・取得

```vb
'===============================================================================
' 追試ファイルを取得（なければ新規作成）
' 戻り値：追試ファイルの Workbook オブジェクト
' 説明：
'   - 既に開いていればそのまま返す
'   - ファイルが存在すれば開いて返す
'   - ファイルが存在しなければ新規作成して返す
'   - 新規作成時は MENU シートを初期化する
'===============================================================================
Public Function GetOrCreateRetestWorkbook() As Workbook
    On Error GoTo ErrorHandler

    Dim filePath As String
    Dim wb As Workbook
    Dim fso As Object

    filePath = GetRetestFilePath()
    Set fso = CreateObject("Scripting.FileSystemObject")

    ' 既に開いているか確認
    Dim fileName As String
    fileName = fso.GetFileName(filePath)
    On Error Resume Next
    Set wb = Workbooks(fileName)
    On Error GoTo ErrorHandler

    If Not wb Is Nothing Then
        Set GetOrCreateRetestWorkbook = wb
        Exit Function
    End If

    ' ファイルが存在する場合は開く
    If fso.FileExists(filePath) Then
        Set wb = Workbooks.Open(filePath)
        Set GetOrCreateRetestWorkbook = wb
        Exit Function
    End If

    ' 新規作成
    Set wb = Workbooks.Add
    Call InitializeRetestMenuSheet(wb)
    
    Application.DisplayAlerts = False
    wb.SaveAs fileName:=filePath, FileFormat:=xlOpenXMLWorkbookMacroEnabled
    Application.DisplayAlerts = True

    Set GetOrCreateRetestWorkbook = wb
    Set fso = Nothing
    Exit Function

ErrorHandler:
    Set GetOrCreateRetestWorkbook = Nothing
    Set fso = Nothing
End Function
```

### 3.4 追試シートの作成

```vb
'===============================================================================
' 追試シートを作成
' 説明：PostingModule.Posting() から呼び出される
'       本体ファイルにテストデータを登録した直後に呼ばれる
' 引数：
'   numTest - 今回登録したテスト数
'   lastRow - テスト入力シートの最終行
'   lastColData - TransferData 呼び出し前のデータシート最終列
'                 （= 新しいテストの列は lastColData + 1 ～ lastColData + numTest）
'===============================================================================
Public Sub CreateRetestSheet(ByVal numTest As Long, ByVal lastRow As Long, _
                              ByVal lastColData As Long)
    On Error GoTo ErrorHandler

    Dim wb As Workbook
    Dim ws As Worksheet
    Dim i As Long, j As Long
    Dim childCount As Long
    Dim testKey As String
    Dim sheetName As String
    Dim method As String
    Dim param As Variant
    Dim targetCol As Long

    ' 追試ファイルを取得
    Set wb = GetOrCreateRetestWorkbook()
    If wb Is Nothing Then
        Call ErrorHandlerModule.ShowValidationError("追試ファイルの作成に失敗しました。")
        Exit Sub
    End If

    childCount = sh_namelist.Range(RNG_NAMELIST_CHILDCOUNT).Value
    method = sh_input.Range("算出方法セル").Value
    param = sh_input.Range("パラメータセル").Value

    ' テスト数分のシートを作成
    For i = 1 To numTest
        targetCol = lastColData + i
        testKey = Sh_data.Cells(eRowData.rowKey, targetCol).Value

        ' シート名の生成（キー_テスト名_観点、31文字制限）
        sheetName = testKey & "_" & _
                    Sh_data.Cells(eRowData.rowTestName, targetCol).Value & "_" & _
                    Sh_data.Cells(eRowData.rowPerspective, targetCol).Value
        If Len(sheetName) > 31 Then
            sheetName = Left(sheetName, 31)
        End If

        ' 同名シートの重複回避
        sheetName = GetUniqueSheetName(wb, sheetName)

        ' シート追加
        Set ws = wb.Sheets.Add(After:=wb.Sheets(wb.Sheets.Count))
        ws.Name = sheetName

        ' ヘッダー情報の書き込み
        With ws
            .Cells(1, 1).Value = "追試管理シート"
            .Cells(1, 1).Font.Size = 12
            .Cells(1, 1).Font.Bold = True

            ' テスト情報ラベル
            .Cells(3, 1).Value = "追試元キー"
            .Cells(4, 1).Value = "教科"
            .Cells(5, 1).Value = "テスト名"
            .Cells(6, 1).Value = "観点"
            .Cells(7, 1).Value = "詳細"
            .Cells(8, 1).Value = "配点"

            ' テスト情報の値
            .Range(RNG_RT_PARENT_KEY).Value = testKey
            .Range(RNG_RT_SUBJECT).Value = Sh_data.Cells(eRowData.rowSubject, targetCol).Value
            .Range(RNG_RT_TEST_NAME).Value = Sh_data.Cells(eRowData.rowTestName, targetCol).Value
            .Range(RNG_RT_PERSPECTIVE).Value = Sh_data.Cells(eRowData.rowPerspective, targetCol).Value
            .Range(RNG_RT_DETAIL).Value = Sh_data.Cells(eRowData.rowDetail, targetCol).Value
            .Range(RNG_RT_ALLOCATE).Value = Sh_data.Cells(eRowData.rowAllocationScore, targetCol).Value

            ' 算出設定ラベル
            .Cells(3, 5).Value = "算出方法"
            .Cells(4, 5).Value = "パラメータ"
            .Cells(5, 5).Value = "状態"

            ' 算出設定の値
            .Range(RNG_RT_METHOD).Value = method
            If Trim(param & "") <> "" Then
                .Range(RNG_RT_PARAM).Value = param
            End If
            .Range(RNG_RT_STATUS).Value = "追試中"

            ' テーブル見出し
            .Cells(RT_HEADER_ROW, RT_COL_CODE).Value = "コード"
            .Cells(RT_HEADER_ROW, RT_COL_LASTNAME).Value = "姓"
            .Cells(RT_HEADER_ROW, RT_COL_FIRSTNAME).Value = "名"
            .Cells(RT_HEADER_ROW, RT_COL_ORIGINAL).Value = "本試"
            .Cells(RT_HEADER_ROW, RT_COL_RETEST_START).Value = "追試1"

            ' 最終得点列（追試1の右）
            Dim finalCol As Long
            finalCol = RT_COL_RETEST_START + RT_COL_FINAL_OFFSET
            .Cells(RT_HEADER_ROW, finalCol).Value = "最終"
            .Cells(RT_HEADER_ROW, finalCol).Font.Bold = True

            ' 児童データの転記
            For j = 1 To childCount
                .Cells(RT_DATA_START_ROW + j - 1, RT_COL_CODE).Value = _
                    Sh_data.Cells(eRowData.rowChildStart + j - 1, eColData.colCode).Value
                .Cells(RT_DATA_START_ROW + j - 1, RT_COL_LASTNAME).Value = _
                    Sh_data.Cells(eRowData.rowChildStart + j - 1, eColData.colLastName).Value
                .Cells(RT_DATA_START_ROW + j - 1, RT_COL_FIRSTNAME).Value = _
                    Sh_data.Cells(eRowData.rowChildStart + j - 1, eColData.colFirstName).Value

                ' 本試の得点（sh_input から取得、TransferData前のデータ）
                Dim originalScore As Variant
                originalScore = sh_input.Cells(eRowInput.rowChildStart + j - 1, _
                                eColInput.colDataStart + i - 1).Value
                If Trim(originalScore & "") = "" Or originalScore = "-" Then
                    .Cells(RT_DATA_START_ROW + j - 1, RT_COL_ORIGINAL).Value = originalScore
                Else
                    .Cells(RT_DATA_START_ROW + j - 1, RT_COL_ORIGINAL).Value = originalScore
                End If

                ' 最終得点の数式を設定
                Call SetFinalScoreFormula(ws, RT_DATA_START_ROW + j - 1, _
                                          RT_COL_ORIGINAL, RT_COL_RETEST_START, finalCol, method)
            Next j

            ' 見出し行の書式
            .Range(.Cells(RT_HEADER_ROW, 1), .Cells(RT_HEADER_ROW, finalCol)).Font.Bold = True
        End With

        ' MENUシートに追加
        Call AddToRetestMenu(wb, testKey, _
            Sh_data.Cells(eRowData.rowSubject, targetCol).Value, _
            Sh_data.Cells(eRowData.rowTestName, targetCol).Value, _
            Sh_data.Cells(eRowData.rowPerspective, targetCol).Value, _
            sheetName)
    Next i

    wb.Save

    Call ErrorHandlerModule.ShowSuccess( _
        "追試シートを作成しました。" & vbCrLf & _
        "追試ファイル: " & wb.Name)

    Exit Sub

ErrorHandler:
    Dim errInfo As ErrorInfo
    errInfo = ErrorHandlerModule.CreateErrorInfo("RetestModule", "CreateRetestSheet")
    Call ErrorHandlerModule.ShowError(errInfo)
End Sub
```

### 3.5 最終得点の数式設定

```vb
'===============================================================================
' 最終得点列に算出方法に応じた数式を設定
' 引数：
'   ws - 追試シート
'   dataRow - 対象行
'   originalCol - 本試列
'   retestStartCol - 追試開始列
'   finalCol - 最終列
'   method - 算出方法
' 説明：
'   追試列が追加されるたびにこの関数を再呼出しして数式を更新する
'   免除 "-" の児童はそのまま "-" を返す
'   追試を受けていない児童は本試の値をそのまま使う
'===============================================================================
Public Sub SetFinalScoreFormula(ByVal ws As Worksheet, ByVal dataRow As Long, _
                                 ByVal originalCol As Long, ByVal retestStartCol As Long, _
                                 ByVal finalCol As Long, ByVal method As String)
    Dim colLetterOrig As String
    Dim colLetterRetestStart As String
    Dim colLetterRetestEnd As String
    Dim colLetterFinal As String
    Dim paramValue As Double
    Dim lastRetestCol As Long
    Dim formula As String

    colLetterOrig = PostingModule.ColumnIndexToLetter(originalCol)
    colLetterRetestStart = PostingModule.ColumnIndexToLetter(retestStartCol)

    ' 最後の追試列 = finalCol - 1
    lastRetestCol = finalCol - 1
    colLetterRetestEnd = PostingModule.ColumnIndexToLetter(lastRetestCol)

    ' 免除（"-"）チェック付きの数式
    ' 本試が "-" ならそのまま "-"
    ' 追試列が全て空欄なら本試の値を使用

    Select Case method
        Case RT_METHOD_MAX
            ' MAX(本試, 追試1, 追試2, ...)
            formula = "=IF(" & colLetterOrig & dataRow & "=""-"",""-""," & _
                      "MAX(" & colLetterOrig & dataRow & "," & _
                      colLetterRetestStart & dataRow & ":" & colLetterRetestEnd & dataRow & "))"

        Case RT_METHOD_INTERPOLATION
            ' α × MAX(全回) + (1-α) × 本試
            paramValue = ws.Range(RNG_RT_PARAM).Value
            formula = "=IF(" & colLetterOrig & dataRow & "=""-"",""-""," & _
                      "ROUND(" & paramValue & "*MAX(" & colLetterOrig & dataRow & "," & _
                      colLetterRetestStart & dataRow & ":" & colLetterRetestEnd & dataRow & ")+" & _
                      (1 - paramValue) & "*" & colLetterOrig & dataRow & ",1))"

        Case RT_METHOD_CAPPED
            ' MAX(本試, MIN(MAX(追試1,追試2,...), 上限点))
            paramValue = ws.Range(RNG_RT_PARAM).Value
            formula = "=IF(" & colLetterOrig & dataRow & "=""-"",""-""," & _
                      "MAX(" & colLetterOrig & dataRow & "," & _
                      "MIN(MAX(" & colLetterRetestStart & dataRow & ":" & _
                      colLetterRetestEnd & dataRow & ")," & paramValue & ")))"

        Case Else
            ' デフォルト：最大値
            formula = "=IF(" & colLetterOrig & dataRow & "=""-"",""-""," & _
                      "MAX(" & colLetterOrig & dataRow & "," & _
                      colLetterRetestStart & dataRow & ":" & colLetterRetestEnd & dataRow & "))"
    End Select

    ws.Cells(dataRow, finalCol).Formula = formula
End Sub
```

### 3.6 追試回の追加

```vb
'===============================================================================
' 追試回を追加
' 引数：ws - 追試シート
' 説明：
'   追試シート上で呼び出す（ボタン or 手動実行）
'   新しい追試列を挿入し、最終列の数式を更新する
'===============================================================================
Public Sub AddRetestRound(ByVal ws As Worksheet)
    On Error GoTo ErrorHandler

    Dim finalCol As Long
    Dim newRetestCol As Long
    Dim childCount As Long
    Dim i As Long
    Dim method As String
    Dim retestNum As Long

    ' 状態チェック
    If ws.Range(RNG_RT_STATUS).Value <> "追試中" Then
        Call ErrorHandlerModule.ShowValidationError( _
            "このテストは既に完了しています。追試回の追加はできません。")
        Exit Sub
    End If

    ' 最終列の位置を検索（ヘッダー行で「最終」を探す）
    finalCol = 0
    Dim col As Long
    For col = RT_COL_RETEST_START To ws.Cells(RT_HEADER_ROW, Columns.Count).End(xlToLeft).Column
        If ws.Cells(RT_HEADER_ROW, col).Value = "最終" Then
            finalCol = col
            Exit For
        End If
    Next col

    If finalCol = 0 Then
        Call ErrorHandlerModule.ShowValidationError("最終列が見つかりません。シート構造が不正です。")
        Exit Sub
    End If

    ' 新しい追試列を最終列の手前に挿入
    ws.Columns(finalCol).Insert Shift:=xlToRight
    newRetestCol = finalCol
    finalCol = finalCol + 1  ' 最終列が1つ右にずれる

    ' 追試番号を設定
    retestNum = newRetestCol - RT_COL_RETEST_START + 1
    ws.Cells(RT_HEADER_ROW, newRetestCol).Value = "追試" & retestNum

    ' 児童数を取得
    childCount = 0
    For i = RT_DATA_START_ROW To ws.Cells(Rows.Count, RT_COL_CODE).End(xlUp).Row
        If Trim(ws.Cells(i, RT_COL_CODE).Value & "") <> "" Then
            childCount = childCount + 1
        End If
    Next i

    ' 最終列の数式を更新（範囲が広がったため）
    method = ws.Range(RNG_RT_METHOD).Value
    For i = 1 To childCount
        Call SetFinalScoreFormula(ws, RT_DATA_START_ROW + i - 1, _
                                  RT_COL_ORIGINAL, RT_COL_RETEST_START, finalCol, method)
    Next i

    Call ErrorHandlerModule.ShowSuccess("追試" & retestNum & " の列を追加しました。")

    Exit Sub

ErrorHandler:
    Dim errInfo As ErrorInfo
    errInfo = ErrorHandlerModule.CreateErrorInfo("RetestModule", "AddRetestRound")
    Call ErrorHandlerModule.ShowError(errInfo)
End Sub
```

### 3.7 追試の完了・本体ファイルへの反映

```vb
'===============================================================================
' 追試を完了し、本体ファイルに結果を反映
' 引数：ws - 追試シート
' 説明：
'   1. 全児童の最終列に値があるか確認
'   2. 本体ファイルのデータシートの該当列の "N" を上書き
'   3. 状態を「完了」→「反映済み」に変更
'   4. MENUシートの状態も更新
'===============================================================================
Public Sub CompleteRetest(ByVal ws As Worksheet)
    On Error GoTo ErrorHandler

    Dim parentKey As String
    Dim childCount As Long
    Dim i As Long
    Dim finalCol As Long
    Dim targetCol As Long
    Dim finalScore As Variant
    Dim unresolvedCount As Long
    Dim mainWb As Workbook

    ' 処理開始
    Call ErrorHandlerModule.BeginProcess

    ' 状態チェック
    If ws.Range(RNG_RT_STATUS).Value = "反映済み" Then
        Call ErrorHandlerModule.ShowInfo("このテストは既に本体ファイルに反映済みです。")
        GoTo CleanExit
    End If

    parentKey = ws.Range(RNG_RT_PARENT_KEY).Value

    ' 児童数を取得
    childCount = 0
    For i = RT_DATA_START_ROW To ws.Cells(Rows.Count, RT_COL_CODE).End(xlUp).Row
        If Trim(ws.Cells(i, RT_COL_CODE).Value & "") <> "" Then
            childCount = childCount + 1
        End If
    Next i

    ' 最終列を検索
    finalCol = 0
    Dim col As Long
    For col = RT_COL_RETEST_START To ws.Cells(RT_HEADER_ROW, Columns.Count).End(xlToLeft).Column
        If ws.Cells(RT_HEADER_ROW, col).Value = "最終" Then
            finalCol = col
            Exit For
        End If
    Next col

    If finalCol = 0 Then
        Call ErrorHandlerModule.ShowValidationError("最終列が見つかりません。")
        GoTo CleanExit
    End If

    ' 未解決チェック（最終列が空欄 or エラーの児童がいないか）
    unresolvedCount = 0
    For i = 1 To childCount
        finalScore = ws.Cells(RT_DATA_START_ROW + i - 1, finalCol).Value
        ' 免除 "-" は解決済みとみなす
        If Trim(finalScore & "") = "" Or IsError(finalScore) Then
            unresolvedCount = unresolvedCount + 1
        End If
    Next i

    If unresolvedCount > 0 Then
        If Not ErrorHandlerModule.ShowConfirmation( _
            "最終得点が未確定の児童が " & unresolvedCount & " 名います。" & vbCrLf & _
            "（追試を受けていない児童は本試の得点が使われます）" & vbCrLf & vbCrLf & _
            "このまま完了しますか？", "未確定の児童がいます") Then
            GoTo CleanExit
        End If
    End If

    ' 確認ダイアログ
    If Not ErrorHandlerModule.ShowConfirmation( _
        "追試結果を本体ファイルに反映します。" & vbCrLf & vbCrLf & _
        "テスト: " & parentKey & " (" & ws.Range(RNG_RT_TEST_NAME).Value & ")" & vbCrLf & _
        "算出方法: " & ws.Range(RNG_RT_METHOD).Value & vbCrLf & vbCrLf & _
        "よろしいですか？", "追試完了確認") Then
        GoTo CleanExit
    End If

    ' 本体ファイルを取得（既に開いていればそのまま、閉じていれば開く）
    Set mainWb = GetMainWorkbook()
    If mainWb Is Nothing Then
        Call ErrorHandlerModule.ShowValidationError( _
            "本体ファイルが見つかりません。" & vbCrLf & _
            "本体ファイルを開いた状態で再度実行してください。")
        GoTo CleanExit
    End If

    ' 本体ファイルのデータシートで該当列を検索
    Dim mainDataSheet As Worksheet
    Set mainDataSheet = mainWb.Sheets("データ")

    targetCol = 0
    Dim lastCol As Long
    lastCol = mainDataSheet.Cells(eRowData.rowKey, Columns.Count).End(xlToLeft).Column
    For col = eColData.colDataStart To lastCol
        If mainDataSheet.Cells(eRowData.rowKey, col).Value = parentKey Then
            targetCol = col
            Exit For
        End If
    Next col

    If targetCol = 0 Then
        Call ErrorHandlerModule.ShowValidationError( _
            "本体ファイルにテストキー「" & parentKey & "」が見つかりません。")
        GoTo CleanExit
    End If

    ' データシートの保護を一時解除
    On Error Resume Next
    mainDataSheet.Unprotect
    On Error GoTo ErrorHandler

    ' 最終得点を本体ファイルに反映
    For i = 1 To childCount
        finalScore = ws.Cells(RT_DATA_START_ROW + i - 1, finalCol).Value

        ' 本体データシートで該当児童の行を特定（コードで照合）
        Dim childCode As String
        childCode = CStr(ws.Cells(RT_DATA_START_ROW + i - 1, RT_COL_CODE).Value)

        Dim targetRow As Long
        targetRow = 0
        Dim k As Long
        For k = eRowData.rowChildStart To eRowData.rowChildStart + childCount - 1
            If CStr(mainDataSheet.Cells(k, eColData.colCode).Value) = childCode Then
                targetRow = k
                Exit For
            End If
        Next k

        If targetRow > 0 Then
            mainDataSheet.Cells(targetRow, targetCol).Value = finalScore
        End If
    Next i

    ' 保護を再設定（本体ファイルのモジュールを呼び出す）
    ' ※追試ファイルから直接呼べないため、Application.Run を使用
    On Error Resume Next
    Application.Run "'" & mainWb.Name & "'!DataManagementModule.ProtectScoreCells"
    On Error GoTo ErrorHandler

    ' 状態を更新
    ws.Range(RNG_RT_STATUS).Value = "反映済み"

    ' MENUシートの状態も更新
    Call UpdateRetestMenuStatus(ws.Parent, parentKey, "反映済み")

    mainWb.Save

    Call ErrorHandlerModule.ShowSuccess( _
        "追試結果を本体ファイルに反映しました。" & vbCrLf & _
        "テスト: " & parentKey)

CleanExit:
    Call ErrorHandlerModule.EndProcess
    Exit Sub

ErrorHandler:
    Call ErrorHandlerModule.CleanupOnError
    Dim errInfo As ErrorInfo
    errInfo = ErrorHandlerModule.CreateErrorInfo("RetestModule", "CompleteRetest")
    Call ErrorHandlerModule.ShowError(errInfo)
End Sub
```

### 3.8 本体ファイルの取得

```vb
'===============================================================================
' 本体ファイルの Workbook を取得
' 説明：
'   追試ファイルから本体ファイルを参照するために使用
'   ファイル名から追試サフィックスを除いた名前で検索する
' 戻り値：本体ファイルの Workbook（見つからなければ Nothing）
'===============================================================================
Public Function GetMainWorkbook() As Workbook
    On Error GoTo ErrorHandler

    Dim fso As Object
    Set fso = CreateObject("Scripting.FileSystemObject")

    Dim retestFileName As String
    Dim mainFileName As String

    retestFileName = fso.GetBaseName(ThisWorkbook.FullName)

    ' サフィックスを除去して本体ファイル名を推定
    If Right(retestFileName, Len(RETEST_FILE_SUFFIX)) = RETEST_FILE_SUFFIX Then
        mainFileName = Left(retestFileName, Len(retestFileName) - Len(RETEST_FILE_SUFFIX))
    Else
        ' このファイル自体が本体ファイルの場合
        Set GetMainWorkbook = ThisWorkbook
        Exit Function
    End If

    ' 開いているワークブックから検索
    Dim wb As Workbook
    For Each wb In Workbooks
        If fso.GetBaseName(wb.Name) = mainFileName Then
            Set GetMainWorkbook = wb
            Set fso = Nothing
            Exit Function
        End If
    Next wb

    ' 見つからない場合、同じフォルダから開く
    Dim mainFilePath As String
    mainFilePath = fso.GetParentFolderName(ThisWorkbook.FullName) & "\" & mainFileName & ".xlsm"

    If fso.FileExists(mainFilePath) Then
        Set GetMainWorkbook = Workbooks.Open(mainFilePath)
    Else
        Set GetMainWorkbook = Nothing
    End If

    Set fso = Nothing
    Exit Function

ErrorHandler:
    Set GetMainWorkbook = Nothing
End Function
```

## 4. 追試ファイル側の構成

### 4.1 ファイル構成

追試ファイル（`成績処理_追試.xlsm`）は本体ファイルから動的に生成される `.xlsm` ファイル。

| シート | 説明 |
|-------|------|
| MENU | 追試管理の一覧。状態・残り人数の表示。シートへのジャンプ機能 |
| （テストごとのシート） | テストキー+テスト名+観点で命名。本試得点・追試得点・最終得点を管理 |

### 4.2 追試ファイルに必要なVBAコード

追試ファイルはコードなしの `.xlsm` として生成し、ボタン操作は **本体ファイルのRetestModule経由で Application.Run を使って実行する** か、あるいは追試ファイル自体にも最小限のVBAコードを埋め込む。

#### 方針：追試ファイルへの最小限コード埋め込み

追試ファイル生成時に、以下のシン・ラッパーコードを `ThisWorkbook` に埋め込む。
ただし **VBAプロジェクトへの書き込みはセキュリティ設定に依存する** ため、代替方針も用意する。

#### 代替方針（推奨）：本体ファイルからの操作

追試ファイルの操作はすべて本体ファイルのVBAから行う。
具体的には以下のフローとする。

1. **追試への得点入力** → 追試ファイルを直接開いて手入力（VBA不要）
2. **追試回の追加** → 本体ファイルのマクロを実行（引数でシートを指定）
3. **完了・反映** → 本体ファイルのマクロを実行

本体ファイル側に追試操作用のUIを追加する（後述の4.3参照）。

### 4.3 本体ファイルへの追試操作UI

本体ファイルのデータシート、またはMENUシートに以下のボタンを追加する。

| ボタン | 処理 | 説明 |
|-------|------|------|
| 追試ファイルを開く | `RetestModule.OpenRetestFile` | 追試ファイルを開く |
| 追試回を追加 | `RetestModule.AddRetestRoundUI` | 対象テストを選択して追試列を追加 |
| 追試を完了 | `RetestModule.CompleteRetestUI` | 対象テストを選択して結果を反映 |

```vb
'===============================================================================
' 追試ファイルを開く
'===============================================================================
Public Sub OpenRetestFile()
    On Error GoTo ErrorHandler

    Dim filePath As String
    filePath = GetRetestFilePath()

    Dim fso As Object
    Set fso = CreateObject("Scripting.FileSystemObject")

    If Not fso.FileExists(filePath) Then

            "追試ありのテストを登録すると自動的に作成されます。")
        Exit Sub
    End If

    Workbooks.Open filePath
    Set fso = Nothing
    Exit Sub

ErrorHandler:
    Dim errInfo As ErrorInfo
    errInfo = ErrorHandlerModule.CreateErrorInfo("RetestModule", "OpenRetestFile")
    Call ErrorHandlerModule.ShowError(errInfo)
```

## 6. 算出方法の仕様

### 6.1 最大値（RT_METHOD_MAX）

```
最終得点 = MAX(本試, 追試1, 追試2, ...)
```

- パラメータ：不要
- 説明：全回の中で最も高い得点を最終得点とする

### 6.2 内分点（RT_METHOD_INTERPOLATION）

```
最終得点 = α × MAX(全回) + (1 - α) × 本試
```

- パラメータ：α（0〜1の実数）
- α = 1 のとき → 最大値方式と同じ
- α = 0 のとき → 本試の値がそのまま使われる
- 例：α = 0.7、本試60点、追試85点の場合
  - `0.7 × 85 + 0.3 × 60 = 59.5 + 18 = 77.5`

### 6.3 追試上限付き（RT_METHOD_CAPPED）

```
最終得点 = MAX(本試, MIN(MAX(追試1, 追試2, ...), 上限点))
```

- パラメータ：上限点（整数）
- 説明：追試の得点には上限を設け、本試より高い場合のみ採用
- 例：上限80点、本試60点、追試95点の場合
  - `MAX(60, MIN(95, 80)) = MAX(60, 80) = 80`

## 7. エラー処理・エッジケース

### 7.1 免除「-」の児童

- 本試が「-」の児童は、追試シートの最終列でも「-」を返す
- 追試列は空欄のままでよい

### 7.2 追試を受けない児童

- 追試列が空欄の場合、最終列の数式で本試の値がそのまま使われる
- MAX関数は空欄を0として扱うため、空欄セルへの対処が必要
  - 数式内で `IF(追試列="", 本試, MAX(本試, 追試))` のようにする
  - または追試を受けない児童には明示的に本試と同じ値を入力する運用でもよい

### 7.3 転入・転出

- 転入児童：追試シート作成時に存在しなかった児童は、追試シートには含まれない
  - 必要に応じて手動で行を追加する運用とする
- 転出児童：追試シートに残るが、本体ファイルの名簿で在籍終了日が設定されていれば反映時にスキップ
  - 実装では反映時にコードで照合するため、名簿にいない児童は無視される

### 7.4 本体ファイルが閉じている場合

- CompleteRetest 実行時に本体ファイルが閉じていれば自動的に開く
- 本体ファイルが見つからない場合はエラーメッセージを表示して中断

### 7.5 追試ファイルの削除・破損

- 追試ファイルが存在しない場合、必要になった時点で再作成する
- 本体ファイルの「N」マーカーは残り続けるため、追試ファイルを再作成して手動で得点を入力し直す
